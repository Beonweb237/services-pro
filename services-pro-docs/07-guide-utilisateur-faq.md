## 1. CRÉER UN COMPTE CLIENT

### 1.1 Inscription étape par étape

**Étape 1** : Allez sur www.servicespro.cm et cliquez sur **"S'inscrire"** en haut à droite.

**Étape 2** : Choisissez **"Je cherche un pro"**.

**Étape 3** : Remplissez le formulaire :
- **Email** : Une adresse valide que vous consultez régulièrement
- **Téléphone** : Votre numéro au format +237XXXXXXXXX
- **Mot de passe** : Au moins 8 caractères, 1 majuscule et 1 chiffre
- **Confirmer le mot de passe** : Identique au précédent

**Étape 4** : Cochez la case **"J'accepte les Conditions d'utilisation"** et cliquez sur **"Continuer"**.

**Étape 5** : Vérification téléphone. Un code à 4 chiffres vous est envoyé par SMS. Saisissez-le dans le champ prévu. Si vous ne recevez pas le SMS sous 2 minutes, cliquez sur **"Renvoyer le code"**.

**Étape 6** : Complétez votre profil (optionnel mais recommandé) :
- Photo de profil
- Votre ville
- Votre quartier

**C'est fait !** Vous êtes maintenant connecté et pouvez rechercher des professionnels.

### 1.2 Astuces

- Utilisez un mot de passe unique que vous n'utilisez nulle part ailleurs
- Votre numéro de téléphone ne sera visible que par les professionnels que vous contactez
- Vous pouvez modifier vos informations plus tard dans "Paramètres"

## 2. RECHERCHER UN PROFESSIONNEL

### 2.1 Recherche rapide

**Depuis la page d'accueil** :
1. Saisissez ce que vous cherchez dans la barre de recherche centrale
2. Exemples de recherche : "plombier Douala", "coiffeur Yaoundé", "électricien"
3. Appuyez sur **Entrée** ou cliquez sur **"Rechercher"**

**L'autocomplete vous aide** : Dès 2 caractères saisis, des suggestions apparaissent (catégories, villes, pros populaires).

### 2.2 Affiner les résultats

Sur la page de résultats, utilisez les **filtres** à gauche :

| Filtre | À quoi ça sert ? | Exemple |
|--------|-------------------|---------|
| **Catégorie** | Choisir un métier spécifique | Plomberie, Électricité |
| **Ville** | Limiter à votre ville | Douala, Yaoundé |
| **Disponibilité** | Trouver un pro disponible quand vous voulez | Aujourd'hui, Cette semaine |
| **Prix min/max** | Filtrer par budget | 10 000 — 50 000 FCFA |
| **Score minimum** | Qualité minimum exigée | Curseur 0-100 |
| **Vérifiés** | Uniquement les pros vérifiés par Services Pro | Oui/Non |
| **Urgence** | Pros acceptant les missions urgentes | Oui/Non |

### 2.3 Trier les résultats

Utilisez le menu **"Trier par"** :
- **Pertinence** (défaut) : Meilleur score global
- **Meilleure note** : Étoiles moyennes décroissantes
- **Plus d'avis** : Nombre d'avis décroissantes
- **Prix croissant** : Du moins cher au plus cher
- **Plus proche** : Distance géographique

### 2.4 Modes d'affichage

- **Grille** (défaut) : Cards avec photo et info essentielles
- **Liste** : Vue compacte, idéale pour comparer
- **Carte** : Pros affichés sur une carte interactive

### 2.5 Astuces de recherche

- **"Autour de moi"** : Autorisez la géolocalisation pour trouver les pros les plus proches
- Sauvegardez vos recherches fréquentes dans vos favoris
- Les résultats mettent en avant les **Trust Seal** (badge doré) = pros d'excellence

## 4. CONTACTER UN PROFESSIONNEL

### 4.1 Messagerie interne (recommandé)

1. Sur le profil du pro, cliquez sur **"Contacter"**
2. Rédigez votre message en décrivant votre besoin :
   - Type de travail à réaliser
   - Localisation
   - Date souhaitée
   - Budget si vous en avez un
3. Cliquez sur **"Envoyer"**
4. Le pro reçoit votre message et vous répond sous 24h (en moyenne)

**Templates de message** :
> "Bonjour [Nom], j'ai une fuite d'eau sous mon évier de cuisine à [Quartier]. Seriez-vous disponible cette semaine pour intervenir ? Merci !"

> "Bonjour [Nom], je souhaiterais refaire la peinture de mon salon (environ 40m²) à [Adresse]. Pourriez-vous me faire un devis ? Disponible pour un rendez-vous en fin de semaine."

### 4.2 Demande de devis

1. Cliquez sur **"Demander un devis"**
2. Remplissez le formulaire structuré :
   - Type de travail
   - Description détaillée
   - Photos du problème/lieu (si pertinent)
   - Date souhaitée
3. Le pro vous envoie un devis estimatif via la messagerie

### 4.3 Appel téléphonique

1. Cliquez sur **"Appeler"**
2. Votre numéro est masqué : le pro ne voit pas votre vrai numéro
3. Un numéro de passerelle vous met en relation
4. **Avantages** : Direct, rapide pour les urgences

### 4.4 Bonnes pratiques

- Soyez précis dans votre demande (ça évite les allers-retours)
- Envoyez des photos si pertinent (pour un devis plus juste)
- Comparez 2-3 pros avant de vous décider
- Ne donnez jamais vos coordonnées bancaires via la messagerie

## 5. APRÈS LA MISSION : NOTER

### 5.1 Quand et comment noter ?

**24 heures après votre mission**, vous recevez :
- Un email de rappel
- Une notification sur la Plateforme
- Un SMS (optionnel)

Cliquez sur **"Noter [Nom du pro]"** pour accéder au formulaire.

**Vous pouvez aussi noter manuellement** :
1. Allez dans **"Mes missions"**
2. Trouvez la mission terminée
3. Cliquez sur **"Noter"**

### 5.2 Le formulaire de notation

**Étape 1** : Notez les 5 critères sur 5 étoiles chacun

| Critère | Questions à vous poser |
|---------|------------------------|
| ⭐ **Qualité du travail** | Le résultat correspond-il à vos attentes ? Est-ce bien fait ? |
| ⭐ **Ponctualité** | Est-il arrivé à l'heure ? A-t-il respecté les délais ? |
| ⭐ **Communication** | A-t-il bien expliqué ? Était-il réactif aux messages ? |
| ⭐ **Rapport qualité/prix** | Le prix était-il justifié par la qualité ? |
| ⭐ **Professionnalisme** | Était-il courtois ? A-t-il respecté vos lieux ? |

**Étape 2** : Rédigez un commentaire (minimum 30 caractères)

> 💡 **Conseil** : Soyez factuel et constructif. Décrivez ce qui s'est bien passé et ce qui pourrait être amélioré. Votre avis aide d'autres clients !

**Exemple de bon commentaire** :
> "Jean a fait un excellent travail sur ma salle de bain. Il est arrivé pile à l'heure, m'a expliqué chaque étape, et le résultat est impeccable. Les finitions sont soignées. Seul petit bémol : il a mis un peu plus de temps que prévu (3h au lieu de 2h), mais le résultat vaut le coup. Je recommande !"

**Étape 3** (optionnel) : Ajoutez des photos du travail réalisé

**Étape 4** : Indiquez si vous recommandez ce pro

**Étape 5** : Choisissez si vous voulez publier anonymement

**Étape 6** : Cliquez sur **"Soumettre mon avis"**

### 5.3 Votre avis compte !

Pourquoi déposer un avis est important :
- 🏆 Ça aide les bons pros à monter dans le classement
- ⚠️ Ça alerte sur les pros à éviter
- 🤝 Ça aide d'autres clients comme vous à faire le bon choix
- 📈 Ça permet au pro de s'améliorer

### 5.4 Modifier votre avis

Vous pouvez modifier votre avis dans les **24 heures** suivant sa publication :
1. Allez dans **"Mes avis"**
2. Trouvez l'avis concerné
3. Cliquez sur **"Modifier"**

Après 24h, l'avis est définitif.

## 7. QUESTIONS FRÉQUENTES — CLIENT

### 🔍 Recherche

**Q : Services Pro est-il disponible dans ma ville ?**
R : Services Pro couvre actuellement Douala, Yaoundé et Bafoussam. Nous ouvrons rapidement de nouvelles villes. Inscrivez-vous pour être informé de notre arrivée dans votre ville !

**Q : Puis-je contacter un pro sans créer de compte ?**
R : Non, vous devez créer un compte gratuit pour contacter un professionnel. Cela nous permet de garantir la qualité des échanges et de lutter contre le spam.

**Q : Comment savoir si un pro est fiable ?**
R : Consultez son Pro Score (plus c'est élevé, mieux c'est), ses avis détaillés, et vérifiez s'il a les badges "Vérifié" et "Trust Seal". Lisez attentivement les commentaires des autres clients.

**Q : Puis-je négocier les prix avec un pro ?**
R : Oui, les tarifs affichés sont des fourchettes indicatives. Le prix final se négocie directement avec le professionnel via la messagerie ou par téléphone.

**Q : Que signifie "Pro Score" ?**
R : C'est un score sur 100 calculé à partir de 5 critères : qualité des avis, fiabilité, réactivité, fidélité des clients et ancienneté. Plus il est élevé, plus le pro est recommandé.

### 💬 Contact & Missions

**Q : Qui paye le pro ? Comment ?**
R : Le paiement se fait directement entre vous et le professionnel (hors Plateforme en Phase 1). Services Pro ne gère pas les paiements. Payez en espèces, par Mobile Money ou selon l'accord avec le pro.

**Q : Le pro ne s'est pas présenté à la mission. Que faire ?**
R : Signalez-le immédiatement via le bouton "Signaler un problème" dans la conversation ou contactez notre support à support@servicespro.cm. Nous prendrons des sanctions appropriées.

**Q : Puis-je annuler une mission ?**
R : Oui, tant que la mission n'est pas commencée. Prévenez le pro via la messagerie. Une annulation répétée sans justification peut impacter votre compte.

**Q : Le pro a fait un mauvais travail. Que faire ?**
R : Déposez un avis honnête décrivant votre expérience. Si le problème est grave (surfacturation, travail dangereux), signalez-le via notre système de signalement ou contactez support@servicespro.cm.

### ⭐ Notation

**Q : Dois-je obligatoirement noter après chaque mission ?**
R : Non, mais nous vous encourageons vivement à le faire ! Votre avis aide la communauté entière. C'est gratuit et prend moins de 2 minutes.

**Q : Mon avis est-il anonyme ?**
R : Par défaut, votre prénom et votre photo apparaissent. Vous pouvez cocher "Publier anonymement" pour que votre nom soit masqué. Le pro voit toujours votre identité réelle pour des raisons de sécurité.

**Q : Puis-je modifier ou supprimer mon avis ?**
R : Vous pouvez le modifier dans les 24h suivant la publication. La suppression n'est pas possible (sauf par l'admin en cas de contenu inapproprié).

**Q : Le pro peut-il supprimer mon avis négatif ?**
R : **Non, jamais.** C'est un pilier de Services Pro. Le pro peut uniquement répondre à votre avis. Si vous pensez que votre avis a été injustement traité, contactez-nous.

**Q : Puis-je noter un pro avec qui je n'ai pas travaillé via Services Pro ?**
R : Non. Le système de notation est strictement réservé aux missions réalisées via la Plateforme. Cela garantit l'authenticité des avis.

## 8. CRÉER UN PROFIL PROFESSIONNEL

### 8.1 Inscription (4 étapes)

**Étape 1 — Créer un compte**
1. Allez sur www.servicespro.cm et cliquez **"Devenir Pro"**
2. Choisissez **"Je suis un professionnel"**
3. Remplissez : Email, Téléphone, Mot de passe
4. Vérifiez votre téléphone (code SMS)

**Étape 2 — Votre identité**
1. **Prénom et Nom** (obligatoire)
2. **Photo de profil** (obligatoire) :
   - Visage visible et centré
   - Fond neutre (pas de distractions)
   - Qualité suffisante (pas floue)
   - Sourire professionnel
   - Format carré, min 400×400px
3. **Photo de couverture** (optionnel mais recommandé) :
   - Photo de vos travaux en action
   - Format panoramique 1200×400px
4. **Pièce d'identité** (obligatoire) :
   - CNI recto-verso ou passeport
   - Pour vérification uniquement (confidentielle)

**Étape 3 — Votre activité**
1. **Catégorie** : Choisissez votre métier principal (Plomberie, Électricité, etc.)
2. **Spécialités** : Sélectionnez vos sous-spécialités (ex: "Fuite d'eau", "Installation sanitaire")
3. **Niveau d'expertise** : Débutant / Confirmé / Expert / Maître
4. **Années d'expérience**
5. **Bio** (100-1000 caractères) :
   - Présentez votre parcours
   - Vos spécialités
   - Ce qui vous différencie
   - Exemple : "Plombier passionné avec 8 ans d'expérience à Douala. Spécialisé dans les réparations urgentes et les installations sanitaires. Intervention rapide, travail soigné, garantie sur toutes mes prestations."

**Étape 4 — Localisation & Tarifs**
1. **Ville principale**
2. **Quartier**
3. **Zone d'intervention** (rayon en km)
4. **Fourchette de prix** (min - max en FCFA)
5. **Type de tarification** : Forfait / Horaire / Sur devis
6. **Devis gratuit** : Oui / Non

**Soumission** : Votre profil est envoyé pour validation. Délai : 24-48h.

### 8.2 Astuces pour passer la validation

- Photo de profil professionnelle (pas de selfie, pas de lunettes de soleil)
- Bio complète et sans faute d'orthographe
- Pièce d'identité lisible (tous les coins visibles)
- Informations cohérentes

## 9. CONFIGURER SON PROFIL OPTIMAL

### 9.1 Les 5 éléments qui font la différence

**1. Photo de profil parfaite**
- Cadrage : Tête et épaules
- Éclairage : Lumière naturelle de face
- Fond : Uni, neutre
- Expression : Sourire confiant
- Évitez : Chapeau, lunettes, photo de groupe

**2. Bio qui convertit**
Structure idéale :
```
[Spécialité principale] avec [X] ans d'expérience à [Ville].
Spécialisé en [sous-spécialité 1] et [sous-spécialité 2].
[Élément différenciant : garantie, réactivité, prix compétitifs].
Intervention rapide, devis gratuit.
```

Exemples de bios qui convertissent :
> "Électricien certifié avec 10 ans d'expérience à Douala. Expert en installations neuves et dépannages urgents. Disponible 7j/7 pour les urgences. Devis gratuit et travail garanti 1 an."

> "Coiffeuse passionnée, spécialiste des tresses et locks. 5 ans d'expérience, salon à Akwa. Je me déplace aussi à domicile. Produits naturels, prix doux. Prenez rendez-vous !"

**3. Portfolio de qualité**
- Minimum 5 photos de vos meilleures réalisations
- Photos avant/après quand c'est possible
- Bon éclairage, cadrage soigné
- Légendes décrivant chaque réalisation

**4. Tarifs transparents**
- Indiquez une fourchette réaliste
- Précisez si vous faites des devis gratuits
- Les tarifs trop bas ou trop élevés découragent

**5. Disponibilités à jour**
- Mettez à jour votre calendrier chaque semaine
- Activez "Disponible immédiatement" quand c'est le cas
- Bloquez vos congés

### 9.2 Checklist profil à 100%

- [ ] Photo de profil uploadée
- [ ] Photo de couverture uploadée
- [ ] Bio complète (100+ caractères)
- [ ] Catégorie sélectionnée
- [ ] 3+ spécialités choisies
- [ ] Niveau d'expertise indiqué
- [ ] Années d'expérience renseignées
- [ ] Tarifs définis
- [ ] Ville et quartier renseignés
- [ ] Zone d'intervention définie
- [ ] 5+ photos dans le portfolio
- [ ] Disponibilités configurées
- [ ] Vidéo de présentation (optionnel mais fortement recommandé)

## 10. GÉRER SES DISPONIBILITÉS

### 10.1 Configurer son calendrier

1. Allez dans **"Disponibilités"** dans votre tableau de bord
2. Vue par défaut : Semaine
3. Cliquez sur un jour pour ajouter des créneaux :
   - Heure de début
   - Heure de fin
   - **"Répéter chaque semaine"** (cocher si c'est récurrent)

### 10.2 Astuces de gestion

- **Créneaux récurrents** : Configurez une fois, répétez automatiquement
  - Exemple : "Tous les lundis, 8h-12h et 14h-18h"
- **Bloquer des dates** : Cliquez sur "Bloquer" pour vos congés
- **Mode "Disponible immédiatement"** : À activer quand vous êtes libre
- **Mode "Urgence"** : À activer si vous acceptez les missions sous 24h

### 10.3 Bonnes pratiques

- Mettez à jour vos dispos **au moins une fois par semaine**
- Un calendrier à jour = plus de visibilité dans les résultats
- Le badge "Réactif" est attribué si vous répondez en < 1h en moyenne

## 11. RÉPONDRE AUX DEMANDES

**Tableau de bord → Demandes reçues**

Chaque demande indique :
- Type (devis, contact direct, urgence)
- Description du besoin
- Localisation
- Date souhaitée
- Budget estimé (si renseigné)

### 11.2 Comment répondre ?

**Réponse rapide = Pro Score qui monte !**

**Accepter** :
> "Bonjour [Client], merci pour votre intérêt. Je suis disponible [date]. Je peux me déplacer sur [zone]. Pourriez-vous m'envoyer une photo de [détail] pour que je prépare l'intervention ? Cordialement, [Votre nom]"

> "Bonjour [Client], merci pour votre message. Malheureusement, je ne suis pas disponible à ces dates / Je ne me déplace pas dans cette zone. Je vous souhaite de trouver le pro qu'il vous faut. Cordialement, [Votre nom]"

### 11.3 Objectif de réactivité

| Temps de réponse | Impact sur votre score |
|------------------|------------------------|
| < 1 heure | Excellent (+points) |
| < 4 heures | Bon |
| < 24 heures | Correct |
| > 24 heures | Négatif (-points) |
| Jamais | Pénalité sévère |

## 12. GÉRER SES AVIS

### 12.1 Consulter ses avis

**Tableau de bord → Mes avis**

Pour chaque avis, vous voyez :
- Les 5 critères notés individuellement
- Le commentaire complet
- Les photos (si présentes)
- La recommandation Oui/Non

### 12.2 Répondre à un avis

**Pourquoi répondre ?**
- Ça montre votre professionnalisme
- Ça montre aux futurs clients que vous êtes à l'écoute
- Ça peut compenser un avis négatif bien géré

**Comment répondre ?**
1. Cliquez sur **"Répondre"** sous l'avis
2. Rédigez votre réponse (10-500 caractères)
3. Cliquez sur **"Publier"**

**Réponse à un avis positif** :
> "Merci beaucoup [Prénom] pour votre confiance et ce retour détaillé ! C'était un plaisir de travailler chez vous. N'hésitez pas à me recontacter pour de futurs projets ! 🙏 [Votre nom]"

**Réponse à un avis négatif** (CRUCIAL) :
> "Merci [Prénom] pour votre retour honnête. Je suis sincèrement désolé que la ponctualité n'ait pas été au rendez-vous ce jour-là — j'ai eu un imprévu sur ma mission précédente. J'ai pris des mesures pour mieux gérer mes déplacements désormais. J'espère avoir l'occasion de vous convaincre lors d'une prochaine mission. [Votre nom]"

**Erreurs à éviter** :
- ❌ "C'est faux, le client ment"
- ❌ "Vous êtes trop exigeant"
- ❌ Ignorer l'avis (pas de réponse)
- ✅ Toujours rester professionnel

### 12.3 Signaler un avis

Si vous pensez qu'un avis est faux ou injustifié :
1. Cliquez sur **"Signaler"** sous l'avis
2. Sélectionnez le motif
3. Ajoutez une explication
4. Notre équipe examine sous 24h

R : Vous pouvez décliner une mission sans problème. Cependant, un taux de déclination trop élevé peut impacter votre score de fiabilité.

**Q : Un client peut-il me laisser un avis négatif injustement ?**
R : Oui, mais vous pouvez toujours répondre de manière professionnelle pour donner votre version. Si vous suspectez une fraude, signalez l'avis. Nous modérons tous les signalements.

**Q : Puis-je supprimer un avis négatif ?**
R : **Non.** C'est un principe fondamental de Services Pro. Vous pouvez uniquement répondre. Cette transparence est ce qui crée la confiance.

**Q : Comment obtenir plus d'avis ?**
R : La meilleure méthode est de demander poliment à vos clients satisfaits de déposer un avis après la mission. Ne proposez **jamais** de rémunération en échange.

**Exemple de demande naturelle** :
> "Si vous êtes satisfait de mon travail, n'hésitez pas à laisser un avis sur Services Pro. Ça m'aide beaucoup et ça aide d'autres clients à me trouver. Merci !"

R : Signalez-le immédiatement via le bouton "Signaler". Notre équipe de modération l'examinera sous 24h.

**Q : Comment est calculé mon Pro Score ?**
R : C'est un score composite sur 100 points basé sur : qualité des avis (40%), fiabilité (20%), réactivité (20%), fidélité client (10%) et ancienneté (10%). Plus le détail est dans votre tableau de bord.

**Q : Pourquoi mon score a baissé ?**
R : Les causes possibles :
- Un nouvel avis négatif
- Vous n'avez pas répondu à des demandes
- Vous avez annulé des missions
- Vous êtes inactif depuis plus de 60 jours

**Q : Comment entrer dans le Top 10 ?**
R : Concentrez-vous sur : excellent travail → avis positifs → réactivité → fiabilité. Le score suivra naturellement.

**Q : Le classement est-il payant ?**
R : **Non.** Le classement est 100% basé sur le mérite (Pro Score). Aucun paiement ne permet de monter dans le classement.

**Q : Comment suis-je payé ?**
R : En Phase 1, les paiements se font directement entre vous et le client (espèces, Mobile Money, etc.). Services Pro ne prend aucune commission.

**Q : Dois-je payer pour recevoir des contacts ?**
R : Non, les contacts sont gratuits. Tous les pros ont la même visibilité de base.
